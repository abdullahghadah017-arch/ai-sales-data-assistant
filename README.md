## Overview
This project allows users to ask questions about sales data in natural language and receive intelligent responses.

## Workflow
- Webhook receives the user request
- AI Agent processes the query
- Google Sheets provides the sales data
- Gemini generates the answer
- Response is returned to the user

## Tools & Technologies
- n8n
- Google Sheets
- Google Gemini
- Webhook
